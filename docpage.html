(fr) Retour à l'éditeur

Chaque ligne du code doit se terminer pas un point virgule ;


(fr)Simple sortie // (en)Simple output

print

tapez "print" suivi d'un espace, ensuite incluer votre message, le message supporte le format html pouvant être inclu entre les balises p, puis vous terminerez votre ligne par un point virgule ;

(fr)Variables

var

tapez "var" suivi d'un espace suivi du nom de la variable que vous souhaitez créer suivi d'un espace suivi de la valeur par défaut, valeur réelle, puis vous terminerez votre ligne par un point virgule ;

printv

tapez "printv" suivi du nom d'une variable existante pour en afficher la valeur (variable numérique :: valeur réelle), puis vous terminerez votre ligne par un point virgule ;

setv

tapez "setv" suivi du nom d'une variable existante suivi d'une valeur, pour changer la valeur actuelle de la variable spécifiée par la valeur précisée (variable numérique :: valeur réelle), puis vous terminerez votre ligne par un point virgule ;

inputv

tapez "inputv" suivi du nom d'une variable existante pour demander à l'utilisateur d'insérer une valeur dans la variable (variable numérique :: valeur réelle), puis vous terminerez votre ligne par un point virgule ;
